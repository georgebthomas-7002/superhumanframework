# Images Folder

This folder contains all images for the Superhuman Framework website.

## Current Images

- **george-thomas.jpg** - Professional photo of George B. Thomas (orange hoodie)
  - Used on: Home page hero, About/Founder page

## Adding New Images

1. Save your image file to this folder
2. Reference it in the code as `/img/your-image-name.jpg`
3. Supported formats: JPG, PNG, SVG, WebP

## Image Guidelines

- **Profile Photos**: 800x800px minimum
- **Hero Images**: 1920x1080px recommended
- **Icons/Logos**: SVG preferred for scalability
- **Optimize**: Use tools like TinyPNG to compress before uploading
